# SAP P2P Analytics Dashboard

## Project Overview
This project is a SAP Procure-to-Pay (P2P) Analytics Dashboard built using Python, SQLite, Pandas, Matplotlib, and Plotly Dash.

It analyzes procurement data and provides KPI insights such as:

- Monthly Procurement Spend
- Top Vendors by PO Value
- Spend by Category
- Regional Spend Analysis
- Buyer Performance
- Spend Tier Distribution

## Files Included

- app.py → Main Dashboard Application
- analytics.py → KPI & Chart Logic
- etl_pipeline.py → ETL Process
- p2p_purchase_data.csv → Source Dataset
- requirements.txt → Required Libraries

## Technologies Used

- Python
- Pandas
- SQLite
- Matplotlib
- Plotly Dash
